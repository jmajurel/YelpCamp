# YelpCamp

YelpCamp is a Web application where users can access to a list campgrounds and comments. It uses a full set of fresh new technologies.
Please feel free to visit the app at http://yelpcamp-revolution.herokuapp.com and add new campgrounds.

Yelpcamp is a complete app that uses Back-end and Front End technologies.
It runs on a nodejs server hosted on heroku. 
The application is built using the web framework express and latest backend and front end librairies, 
Bootstrap to font end design
Mongoose to communicate with the MongoBD database
Passportjs to handle user accounts
